q:
cara mencari tahu website mana saja yang memakai math.js

a-ai:
Cara termudah untuk mencari tahu website mana saja yang memakai **math.js** adalah dengan **menggunakan layanan pelacak teknologi web (_tech stack profiler_) atau memeriksa kode sumber website secara manual**.
<br>

Berikut adalah beberapa metode efektif yang bisa Anda gunakan:
<br>

Menggunakan Layanan Tech Stack Profiler (Paling Praktis)
<br>

Layanan ini mengindeks jutaan situs web dan mendeteksi pustaka JavaScript apa saja yang tertanam di dalamnya.
<br>

&bull; **Wappalyzer**: Anda bisa langsung melihat daftar situs yang terdeteksi menggunakan pustaka ini melalui halaman [Websites using math.js di Wappalyzer](https://www.wappalyzer.com/technologies/javascript-libraries/math-js/). Mereka mengelompokkan website berdasarkan tren lalu lintas (traffic), negara, hingga kategori bisnis.
<br>

&bull; **BuiltWith**: Ketik "math.js" atau "mathjs" di kolom pencarian situs [BuiltWith](https://builtwith.com/) untuk melihat tren penggunaan global, grafik statistik, serta daftar nama domain yang aktif memanfaatkan pustaka tersebut.

<br>

Menggunakan Ekstensi Browser (Untuk Cek Satu per Satu)
<br>

Jika Anda sedang membuka suatu website dan penasaran apakah situs tersebut menggunakan [math.js](https://mathjs.org/), pasang ekstensi browser berikut:
<br>

&bull; **Wappalyzer Extension**: Tersedia di Chrome Web Store atau Firefox Add-ons. Ekstensi ini akan memunculkan ikon di *address bar* yang menampilkan semua daftar teknologi (termasuk pustaka JavaScript) dari website yang sedang Anda buka.
<br>

&bull; **Library Detector**: Ekstensi khusus yang langsung mendeteksi pustaka JavaScript yang berjalan di halaman web saat itu juga.

<br>

Memeriksa Secara Manual lewat Inspect Element
<br>

Jika tidak ingin menginstal ekstensi apa pun, Anda bisa memanfaatkan fitur bawaan browser (Chrome, Edge, Firefox):
<br>

1\. Buka website yang ingin Anda periksa.
<br>

2\. Klik kanan di area mana saja, lalu pilih **Inspect** (atau tekan `F12`).
<br>

3\. Masuk ke tab **Console**.
<br>

4\. Ketik perintah berikut lalu tekan Enter:
```
typeof math !== 'undefined' ? "Memakai math.js" : "Tidak memakai math.js"

```
*Catatan: Metode konsol ini berhasil jika developer website mengekspos objek `math` ke ranah global script halaman.*
<br>

5\. Sebagai alternatif, masuk ke tab Sources atau Network, lalu gunakan fitur pencarian (`Ctrl + Shift + F` atau `Cmd + Option + F`) dan ketik kata kunci seperti `math.js`, `math.min.js`, atau link CDN yang mengarah ke [cdnjs mathjs](https://cdnjs.com/libraries/mathjs).

<br>

Jika Anda membutuhkan **data mentah atau laporan spesifik** mengenai industri apa saja yang paling banyak menggunakan pustaka ini, opsi berbayar dari Wappalyzer atau BuiltWith menyediakan unduhan daftar domain tersebut dalam format CSV.
<br>
